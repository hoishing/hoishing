# 👋 Hi~ Kelvin's Speaking 📣

I love coding, writing program is an enlightening✨ creation process. No matter frontend, backend, mobile apps, task automation, machine learning, algo trading... I like them all.

I make my living mainly by creating [iOS apps](https://apps.apple.com/hk/developer/fbm/id371152397), occasionally I do take freelance jobs. However, I spend quite a considerable amount of time(if not the most 😆) trying out new tech / methodology / frameworks. It's quite a hobby of mine, but I normally did it in private without mentioning it, no matter whether that piece of tech is interesting or not.

One day, I saw my beloved Taiwan IT minister [Audrey Tang](https://zh.wikipedia.org/zh-tw/%E5%94%90%E9%B3%B3) recommend a [work out loud](https://youtu.be/XpjNl3Z10uc) methodology. I'm convinced.

Here listed some of my works, both professional and casual ones. Hope it helps connecting us 🌐
