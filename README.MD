# 👋 Hi~ Kelvin's Speaking 📣

I love coding, writing program is an enlightening✨ creation process. No matter frontend, backend, mobile apps, task automation, machine learning, algo trading... I like them all.

I make my living mainly by creating [iOS apps](https://apps.apple.com/hk/developer/fbm/id371152397), occasionally I did take freelance projects. However, I spend quite a considerable amount of time(if not the most 😆) trying out new tech / methodology / frameworks.
